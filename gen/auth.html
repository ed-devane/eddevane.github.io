Step 1: Install ESP32 Board Support

Install through Arduino IDE's Board Manager:

  1. Open Arduino IDE
  2. Click: File > Preferences
  3. Find "Additional Board Manager URLs" field
  4. Paste this URL: https://espressif.github.io/arduino-esp32/package_esp32_index.json
  5. Click OK
  6. IMPORTANT: Log out of Windows and log back in (this applies the JSON file)
  7. Open Arduino IDE again
  8. Click: Tools > Board > Boards Manager (wait for it to load)
  9. In search box, type: esp32
  10. Find "esp32 by Espressif Systems" and click INSTALL
  11. Wait 2-5 minutes for download and installation to complete
  12. When finished, close Boards Manager
Note: If Preferences hangs after pasting the URL, log out and log back in to Windows.

Step 2: Install Soniphorm Gen Board Profile

Add the custom board profile with optimized settings:

  1. Close Arduino IDE (if open)
  2. Press Windows Key + R
  3. Paste: %LOCALAPPDATA%\Arduino15\packages\esp32\hardware\esp32
  4. Press Enter - you'll see a folder with a version number (e.g., 3.0.7 or 3.3.4)
  5. Open that version folder, then open boards.txt in Notepad
  6. Create backup: Right-click > Copy > Paste > Rename to boards.txt.backup
  7. Download board profile file (button below)
  8. Open downloaded soniphorm_board.txt in Notepad
  9. Copy all text (Ctrl+A, Ctrl+C)
  10. Go back to boards.txt, scroll to bottom (Ctrl+End)
  11. Paste at end (Ctrl+V)
  12. Save and close (Ctrl+S)
  13. Restart Arduino IDE
  14. Go to: Tools > Board > esp32 > Soniphorm Gen v1.0
Download Board Profile (soniphorm_board.txt)

Step 1: Install ESP32 Board Support

Install through Arduino IDE's Board Manager:

  1. Open Arduino IDE
  2. Click: Arduino IDE > Settings (or press Cmd + ,)
  3. Find "Additional Board Manager URLs" field
  4. Paste this URL: https://espressif.github.io/arduino-esp32/package_esp32_index.json
  5. Click OK
  6. Click: Tools > Board > Boards Manager
  7. Wait for the index to load (may take 30 seconds)
  8. In search box, type: esp32
  9. Find "esp32 by Espressif Systems" and click INSTALL
  10. Wait 2-5 minutes for download and installation to complete
  11. When finished, close Boards Manager

Step 2: Install Soniphorm Gen Board Profile

Add the custom board profile with optimized settings:

  1. Quit Arduino IDE completely (Cmd + Q)
  2. Open Finder
  3. Press Cmd + Shift + G (Go to Folder)
  4. Paste: ~/Library/Arduino15/packages/esp32/hardware/esp32
  5. Press Enter - you'll see a folder with a version number (e.g., 3.0.7 or 3.3.4)
  6. Open that version folder
  7. Find boards.txt file
  8. Right-click > Duplicate to create backup (boards copy.txt)
  9. Right-click boards.txt > Open With > TextEdit
  10. Download board profile file (button below)
  11. Open downloaded soniphorm_board.txt in TextEdit
  12. Select All (Cmd + A) and Copy (Cmd + C)
  13. Go back to boards.txt, scroll to the very bottom
  14. Paste at end (Cmd + V)
  15. Save (Cmd + S) and close
  16. Open Arduino IDE
  17. Go to: Tools > Board > esp32 > Soniphorm Gen v1.0
Download Board Profile (soniphorm_board.txt)
No external libraries needed! All required drivers (MCP23017, NeoPixel) are now built into the generated code.

Troubleshooting: Can't Find Library Folder?

The Library folder is hidden by default on macOS. Here's how to access it:

  1. Open Finder
  2. Click Go menu in the menu bar
  3. Hold down the Option key - "Library" will appear in the menu
  4. Click Library
  5. Navigate to: Arduino15 > packages > esp32 > hardware > esp32 > [version]

Alternative method: Open Terminal and type:

open ~/Library/Arduino15/packages/esp32/hardware/esp32
