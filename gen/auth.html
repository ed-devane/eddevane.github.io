IMPORTANT: Install through Arduino IDE's Board Manager (not manual download!):

  1. Open Arduino IDE
  2. Click: File > Preferences
  3. Find "Additional Board Manager URLs" field
  4. Paste this URL: https://espressif.github.io/arduino-esp32/package_esp32_index.json
  5. Click OK
  6. Click: Tools > Board > Boards Manager (wait for it to load)
  7. In search box, type: esp32
  8. Find "esp32 by Espressif Systems" and click INSTALL
  9. Wait 2-5 minutes for download and installation to complete
  10. When finished, close Boards Manager

Note: Do NOT manually extract ESP32 files to desktop or other folders. Arduino IDE installs everything automatically to the correct hidden folder.
